Tugas Pemrograman Jaringan - Deteksi Kelas IP
Program Java sederhana untuk mendeteksi Kelas IP, Network ID, Host ID, dan melakukan konversi bilangan biner dari sebuah input URL. Tugas ini dibuat untuk memenuhi penugasan mata kuliah Pemrograman Jaringan.

Fitur Program
Menerima input berupa URL (contoh: www.polinema.ac.id atau www.google.com).
Melakukan DNS Lookup untuk mendapatkan IPv4 Address.
Menentukan Kelas IP Address (A, B, C, D, atau E).
Mengidentifikasi bagian Network ID dan Host ID.
Mengonversi setiap oktet (blok) IP Address menjadi representasi bilangan biner 8-bit.
Cara Menjalankan Program
Pastikan Anda sudah menginstal Java Development Kit (JDK) di komputer.
Buka terminal atau command prompt dan arahkan ke direktori tempat file ini berada.
Compile file Java dengan perintah:
javac DeteksiIP.java
